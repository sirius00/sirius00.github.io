使用技巧：#

快速替换文本：#

ctrl + f ： 搜索文本

shift + command + L： 选择

快速打开文件：

command + P

查看函数定义：

command + 鼠标点击函数名

快捷键#

内置快捷键:#

打开终端: shift + ctrl + `

打开命令窗口: shift + command + p

插件使用#

bookmarks(添加书签)#

打开命令窗口, 输入 bookmarks

添加书签: opt + command + k

project manage(项目管理):#

打开命令窗口, 输入project manage
